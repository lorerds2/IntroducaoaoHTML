Tags Estruturais

Tags O que siginfica
<html>...</html> A tag HTML, abreviação de HyperText Markup Language, é usada para estruturar o conteúdo de uma página da web.
<head>...</head> Essa tag HTML é usada para conter informações sobre o documento HTML que não são exibidas diretamente no navegador.
<title>...</title> Define o título da página, que aparece na barra de título do navegador ou na aba.
<body>...</body> É a tag usada para definir o corpo principal de um documento HTML. Todo o conteúdo visível de uma página web é colocado dentro desta tag.
<div>...</div> A palavra "div" é uma abreviação de "divisão". Essencialmente, a tag é usada para agrupar e organizar outros elementos HTML em seções lógicas ou grupos semânticos.

Tags Semânticas

Tags O que siginfica
<header>...</header> Esta tag é usada para definir o cabeçalho de uma seção ou da página inteira. Geralmente contém elementos como logotipos, títulos, menus de navegação ou outros elementos introdutórios.
<footer>...</footer> Usada para definir o rodapé de uma seção ou da página inteira. Normalmente contém informações como informações de copyright, links para políticas de privacidade, informações de contato ou créditos.
<main>...</main> Esta tag é usada para envolver o conteúdo principal de uma página. Deve conter apenas o conteúdo diretamente relevante para o propósito da página.
<asidearticle>...</asidearticle> Usada para definir um conteúdo independente e autossuficiente que pode ser reutilizado ou distribuído de forma independente, como um post de blog, uma notícia ou um comentário.
<aside>...</aside> Esta tag é usada para definir conteúdo relacionado, mas que está separado do conteúdo principal da página. Pode conter elementos como barras laterais, anúncios, links relacionados ou outros conteúdos adicionais.

Tags de Texto

Tags O que siginfica
<h1...h6></h1...h6> Essas são tags de cabeçalho. <h1> é o maior e mais importante, e <h6> é o menor. Elas são usadas para indicar a importância e a hierarquia do texto.
<hgroup>...</hgroup> Esta tag é usada para fechar um grupo de cabeçalhos (<h1> a <h6>) quando eles estão sendo usados juntos para representar um título ou subtítulo principa.
<p>...</p> Essa tag é usada para fechar um parágrafo de texto. Parágrafos são blocos de texto separados que ajudam a organizar o conteúdo em uma página da web. .
<b>...</b> Esta tag é usada para fechar um trecho de texto em negrito.
<i>...</i> e <s>...</s> A primeira tag é usada para colocar o texto em itálico, a segunda é usada para colocar um texto riscado entre <s> e </s> assim exibido com uma linha horizontal através dele.

Links e âncoras

Tags O que siginfica
<a href=””>Nome da âncora</a> Essa é a tag de âncora. Ela cria um link para outra página da web ou recurso. Você insere o endereço da página ou recurso na parte "href" e o texto que será clicável.
<a href=mailto:> Essa tag cria um link para enviar um e-mail. Ao clicar no link, o programa de e-mail padrão do usuário será aberto com o endereço de e-mail especificado no "href".
<a href=”tel://#####-####”>Número de telefone</a> Esta tag cria um link para discar um número de telefone quando clicado em dispositivos móveis ou em alguns navegadores .
<a name=”name”>...</a> Esta tag é usada para criar um marcador em uma página da web. O atributo "name" define o nome do marcador, que pode ser usado para direcionar os usuários para uma parte específica da página.

Imagens

Tags O que siginfica
</img >src=”URL” Esta tag é usada para incluir uma imagem em uma página da web. O atributo "src" especifica o URL da imagem que será exibida na página.
alt=” texto” Este é um atributo usado com a tag 'img'. Ele fornece um texto alternativo que é exibido se a imagem não puder ser carregada ou se o usuário estiver usando um leitor de tela. Isso ajuda na acessibilidade e na compreensão do conteúdo da imagem.
height altura da imagem .
width largura da imagem.
border=” ...”<área/> O primeiro atributo usado com a tag 'img'. Ele especifica a largura da borda ao redor da imagem, em pixels. O segundo é uma tag usada dentro da tag 'map'. Ela define uma área clicável em uma imagem mapeada usando a tag 'map'.
shape=” ...” Este é um atributo usado com a tag 'área'. Ele especifica a forma da área clicável, que pode ser "rect" (retangular), "circle" (circular) ou "poly" (poligonal) .
coords=” ...” Este é um atributo usado com a tag 'área'. Ele especifica as coordenadas da área clicável, dependendo da forma especificada no atributo "shape"..
<map>...</map> Esta tag é usada para criar um mapa de imagem em uma página da web. Ela envolve as áreas clicáveis definidas pelas tags 'área' em uma imagem. Isso permite que diferentes partes da imagem sejam clicáveis e levem a diferentes destinos..

Listas

Tags O que siginfica
<ol>...</ol> Essa tag é usada para criar uma lista ordenada, onde os itens são numerados automaticamente em ordem sequencial. Cada item da lista é envolvido pela tag .
<ul>...</ul> Esta tag é usada para criar uma lista não ordenada, onde os itens não são numerados. Cada item da lista é envolvido pela tag 'li'.
<li>...</li> : Essa tag é usada dentro das tags 'ol' ou 'ul' para representar cada item da lista, seja ordenada ou não. Ela envolve o conteúdo de cada item da lista.
reversed Este é um atributo usado com a tag 'ol'. Quando presente, inverte a ordem dos números na lista ordenada, começando do maior número para o menor.
start Este é um atributo usado com a tag 'ol'. Ele especifica o número inicial da lista ordenada. Por exemplo, se você definir start="3", a lista começará a numerar a partir do número 3. .
